# 5.5 plan timeline review

## Summary

Team reviewed the updated 5.5 release plan and scope: feature freeze next week, beta in January, and restricted release in March, with discussion on the restricted-to-GA gap. Key focus areas included technical debt/resilience, software-defined e-box enablement, cloud scope (Azure GA in 5.5), Query Engine DDL inclusion (without group-by), and multiple hardware/supply-chain risks requiring risk buys and alternate sourcing. Several follow-ups were captured around Broadcom 4.2 justification, BOM/qualification readiness, GMP POC clarity, and HTX dark-site install timing vs December beta readiness.

## Decisions
- Azure GA is part of 5.5; decommission delivered in 5.5 for cloud flows
- Query Engine DDL included in 5.5; group-by not in 5.5
- SMB scale deferred to 5.6
- SPDK target and DB projections moved to 5.6
- NFSv4 delegations view-level added in 5.5
- Physical quota added in 5.5
- SMB server-side copy (SMBX copy) added in 5.5
- Broadcom 4.2 is not currently in 5.5 scope (requires separate justification)
