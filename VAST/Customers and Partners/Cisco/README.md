## Key Facts

- Cisco request (RFE 0526) focuses on regex-based allow/deny patterns for replication policies.

- Cisco is participating as a partner/sponsor at VAST Forward.

- Cisco manages many global sites with data replication and has hundreds of replication relationships.

- Cisco requires replication policy filtering that can allow or deny files/objects by name using regex patterns (examples: allow *.LIB, deny *.mp3).

- Cisco's value proposition for RFE 0526 includes enterprise multi-geo liability protection by preventing replication of copyrighted content via 'never replicate' patterns.
