## Key Information


- Microsoft AI Infrastructure (MAI) training deployment at the Nebius New Jersey data center is planned to scale to an 812 x 812 configuration at full build-out.

- The MAI Nebius New Jersey deployment is designed to be installed in phases, with the first phase requiring hardware for one hall and a 58 x 58 setup.

- Latest update as of 2026-01-05 is that Microsoft expects to install the first hall of the Nebius New Jersey MAI deployment in the May/June 2026 timeframe, then move aggressively to the other 13 halls before the end of 2026.

- VAST is running a parallel evaluation with Microsoft MAI to determine whether the Nebius New Jersey deployment can be done on Microsoft hardware (Azure Blob or Azure Compute) or using Microsoft NAND supply chain, due to severe supply chain constraints.
